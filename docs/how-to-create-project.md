## Project Settings
Use the -settings option to pass multiple arguments to the template.

```bash
npm create naty@latest -settings <arg1> <arg2> ...
```

An error will occur if the number of arguments is incorrect.

<br>

## Third-party Template
If you use third-party templates, please add this option.

```bash
npm create naty@latest@latest -url <url>
```
Please use the URL in the template.
